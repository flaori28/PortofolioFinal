# Portfolio - OPERATIVE GHOST (BTS SIO)

Un portfolio professionnel au design immersif inspiré de l'interface tactique de **Call of Duty: Black Ops 2**. 
Conçu pour présenter un profil technique de manière futuriste et impactante.

## 🎮 Thème Visuel
- **Style :** Militaire / Futuriste / Cyberpunk 2025
- **Palette :** Dark Grey (`#0b0d10`), BO2 Orange (`#ff9a00`), Cyber Blue (`#00f0ff`)
- **Effets :** 
  - Scanlines & Glitchs texte
  - Particules de fond interactives (Canvas HTML5)
  - Animations HUD & Interfaces tactiques

## 📂 Structure du Projet (Multi-pages)
Le site est divisé en pages statiques distinctes pour une navigation claire.

- `index.html` : **Accueil** - Hero section avec animation d'entrée.
- `apropos.html` : **Profil** - Biographie et informations de contact.
- `tcs.html` : **Parcours** - Timeline des études et expériences.
- `competences.html` : **Arsenal** - Compétences techniques avec barres de progression.
- `projets.html` : **Dossiers** - Présentation des projets réalisés (cartes style dossiers classés).
- `veille.html` : **Renseignement** - Articles de veille technologique.
- `certifications.html` : **Badges** - Certifications obtenues.

## 🚀 Installation & Utilisation
1. Cloner ce dépôt.
2. Ouvrir le dossier dans **Visual Studio Code**.
3. Lancer avec **Live Server** (extension VS Code) ou ouvrir `index.html` dans un navigateur.

## 🛠 Technologies
- **HTML5** : Structure sémantique.
- **CSS3** : Flexbox, Grid, Animations méchaniques & néon variables CSS.
- **JavaScript (Vanilla)** : Gestion du Canvas (particules), effets sonores simulés et glitchs.

## 📝 Personnalisation
Pour modifier les couleurs principales, éditez les variables CSS dans `assets/css/style.css` :
```css
:root {
  --bo2-orange: #ff9a00; /* Couleur d'accentuation principale */
  --bo2-blue: #00f0ff;   /* Couleur secondaire */
}
```
