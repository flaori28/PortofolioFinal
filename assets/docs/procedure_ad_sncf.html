2. Installation Windows serveur

Sélectionner Windows Server 2025 Standard Evaluation avec l'expérience utilisateur pour faciliter la gestion initiale.

Recommandation : Choisir Windows Server 2025 Standard Evaluation (experience utilisateur) pour disposer d'une interface graphique lors de la configuration initiale.
Étapes d'installation :
  1. Sélectionner l'image d'installation Windows Server 2025
  2. Procéder à l'installation standard
  3. À l'écran "Sélectionner une image", choisir "Windows Server 2025 Standard Evaluation (expérience utilisateur)"
  4. Suivre l'assistant d'installation
  5. À l'écran "Paramètres de personnalisation", définir un mot de passe pour l'administrateur

3. Procédure AD1EX configuration

3.1 Activation du bureau à distance

Étapes :
  1. Accéder aux Propriétés système
  2. Aller dans l'onglet "Utilisation à distance"
  3. Activer "Autoriser les connexions à distance à cet ordinateur"
  4. Cocher "N'autoriser que les connexions provenant d'ordinateurs exécutant le Bureau à distance avec authentification NLA (recommandé)"

3.2 Désactivation de la sécurité renforcée d'Internet Explorer

Configuration de la sécurité :
  • Administrateurs : Désactiver la Configuration de sécurité renforcée
  • Utilisateurs : Désactiver la Configuration de sécurité renforcée

3.3 Changement du nom de l'ordinateur

Configuration du nom d'hôte :
  1. Accéder aux Propriétés système
  2. Cliquer sur "Modifier" dans la section "Nom d'ordinateur"
  3. Remplacer le nom par : SRV-AD1EX
  4. Valider et redémarrer la machine
Nom d'ordinateur: SRV-AD1EX
Groupe de travail: WORKGROUP
Description: Serveur de production IS ou "Serveur de gestion"

3.4 Configuration des adresses IP

Configuration réseau statique :
  • Adresse IP: 192.168.17.1
  • Passerelle par défaut: 192.168.17.254
  • Masque de sous-réseau: 255.255.255.0
  • Serveur DNS préféré: 192.168.17.1 (lui-même une fois AD configuré)
Vérification : Utiliser la commande ipconfig pour valider la configuration avant de continuer.

3.5 Désactivation de l'IPv6

Étapes :
  1. Ouvrir Paramètres de la carte Ethernet
  2. Accéder aux Propriétés du Protocole Internet version 4 (TCP/IPv4)
  3. Décocher la case IPv6
  4. Valider les modifications

3.6 Ajout des rôles et fonctionnalités

Rôles à installer :
  • ✓ Serveur DHCP (Installer)
  • ✓ Serveur DNS (Installer)
  • ✓ Services de domaine Active Directory (Installer)
Procédure d'installation des rôles :
  1. Ouvrir le Gestionnaire de serveur
  2. Cliquer sur "Ajouter des rôles et fonctionnalités"
  3. Sélectionner "Installation basée sur un rôle ou une fonctionnalité"
  4. Sélectionner le serveur destination (SRV-AD1EX)
  5. Sélectionner les rôles listés ci-dessus
  6. Ajouter les fonctionnalités requises pour chaque rôle
  7. Cliquer sur "Installer"
[Image: Sélection des rôles de serveur dans le gestionnaire]

3.7 Promotion en contrôleur de domaine

Configuration du domaine :
  1. Cliquer sur l'alerte de configuration post-déploiement
  2. Sélectionner "Promouvoir ce serveur en contrôleur de domaine"
  3. Choisir "Ajouter une nouvelle forêt"
  4. Entrer le nom de domaine : carnoux.local
  5. Cliquer sur "Suivant"
Nom de domaine racine: carnoux.local
Niveau fonctionnel de la forêt: Windows Server 2025
Niveau fonctionnel du domaine: Windows Server 2025
Configuration DNS :
  1. Une alerte s'affichera indiquant que DNS n'existe pas encore (normal)
  2. Laisser l'assistant créer automatiquement la zone DNS
  3. Cliquer sur "Suivant"
Options supplémentaires :
  • NetBIOS du domaine: CARNOUX
  • Répertoires de données: Par défaut (C:\Windows\NTDS)
  • Répertoire des journaux: Par défaut (C:\Windows\NTDS)
  • Répertoire SYSVOL: Par défaut (C:\Windows\SYSVOL)

3.8 Configuration DNS - Zone inversée

Créer une nouvelle zone inversée :
  1. Ouvrir le Gestionnaire DNS (Outils > DNS)
  2. Clic droit sur "Zones de recherche inversée"
  3. Sélectionner "Nouvelle zone"
  4. Choisir "Zone principale"
  5. Sélectionner "Zone de recherche inversée IPv4"
  6. Entrer l'ID du réseau : 192.168.17 (sans la partie hôte)
  7. Activer les mises à jour dynamiques
  8. Cliquer sur "Terminer"

3.9 Configuration DHCP

Créer une nouvelle étendue DHCP :
  1. Ouvrir le Gestionnaire DHCP
  2. Clic droit sur IPv4 > "Nouvelle étendue"
  3. Donner un nom : DHCP
  4. Description : srv-dhcp
Configuration des adresses IP DHCP :
  • Adresse IP de début: 192.168.17.50
  • Adresse IP de fin: 192.168.17.100
  • Masque de sous-réseau: 255.255.255.0
  • Durée du bail: 24 heures (par défaut)
Configuration des paramètres DHCP :
  • Passerelle par défaut: 192.168.17.254
  • Serveur DNS: 192.168.17.1 (et optionnellement 8.8.8.8)
  • Domaine parent: carnoux.local
Activation du serveur DHCP :
  • Cocher "Oui, je veux activer cette étendue maintenant"
  • Cliquer sur "Suivant" et "Terminer"

3.10 Création d'utilisateurs de domaine

Ajouter un nouvel utilisateur :
  1. Ouvrir "Utilisateurs et ordinateurs Active Directory" (Outils > ADUC)
  2. Naviguer vers Users
  3. Clic droit > "Nouvel objet" > "Utilisateur"
  4. Entrer le prénom: SIO
  5. Nom d'ouverture de session: SIO1@carnoux.local
  6. Nom d'ouverture de session (Windows 2000): CARNOUX\SIO1
Configuration du mot de passe :
  • Cocher "L'utilisateur ne peut pas changer de mot de passe"
  • Cocher "Le mot de passe n'expire jamais"
  • Définir un mot de passe sécurisé
